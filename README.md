# Coleção de Consoles

A ideia desse projeto é imaginar que há uma caixa onde o usuário coleciona consoles. 

O primeiro console que ela adiciona ficará em baixo, e os posteriores por cima. 

Tendo assim uma ideia de pilha, onde caso queira retirar algum console da caixa (literal caixa mesmo, como de papelão), seria necessário retirar a mais recente (que estaria mais acima).
